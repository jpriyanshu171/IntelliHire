## IntelliHire Backend

### Run

- **Prerequisites**: Java 17, Maven, PostgreSQL.
- **Database**: create a database named `intellihire` and update credentials in `src/main/resources/application.properties` if needed.
- **Start**:
  - From project root: `cd backend`
  - Run: `mvn spring-boot:run`

Backend will run on `http://localhost:8080`.
